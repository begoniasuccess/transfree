# transfree

## 作品說明
從命名上Trans及Free為發想，希望使用者能自由地暢遊， 配色以明亮、活潑的顏色做為設計，融合星球地標的元素，
表達行動能夠無拘束地到達使用者想去的目的地。
## 系統說明

專案運行方式是使用`npm install`、`npm run serve`
Node.js 版本 v16.13.0
